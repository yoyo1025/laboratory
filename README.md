# laboratory
## open3d-example
### ファイルの読み込み、書き込み
open3d-example/input-outputにて  
**ビルド**
```
docker build -t input-output .
```
**実行**
```
docker run --rm -v "$PWD:/app" open3d-example
```

### ファイルの読み込み、書き込み
open3d-example/meshにて  
**ビルド**
```
docker build -t mesh .
```
**実行**
```
docker run --rm -v "$PWD:/app" open3d-example
```
